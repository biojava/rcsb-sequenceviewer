The org.rcsb.sequence package provides a front-end API to the SequenceCollection data model.

This API is used by the front-end logic that powers the "Sequence Details Report" which may be seen by clicking here.

Example usage 1 -- residues in a sequence: 

// Get a sequence object
Sequence sequence = StructureCollections.get("4HHB").getChain("A");

// Print a string representation of the sequence to stdout
System.out.println( sequence.getSequenceString() );

// Get residue ids from SEQRES record (the default operation)
Collection<ResidueId> seqresRecordIds = sequence.getResidueIds();

// Get residue ids from ATOM records
Collection<ResidueId> atomRecordIds = sequence.getResidueIds(ResidueNumberScheme.ATOM);

Example usage 2 -- getting information about features on a sequence: 

// Get a sequence
Sequence sequence = StructureCollection.get("4HHB").getChain("A");

// Get SCOP annotation
{@link ScopAnnotationGroup} scop = sequence.getAnnotationGroup(ScopAnnotationGroup.class);
